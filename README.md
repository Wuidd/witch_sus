## 🚀 Features

### 🔍 证据搜查
用于案发后、查证环节前布置线索的框架。  

- 🪧 **线索机制**  
  - 线索是留在各个方块中不可见的数据  
  - 玩家需通过 **放大镜** 的提示来寻找这些隐形线索  
  - 锁定到有线索的方块后，使用放大镜右击即可打开容器界面  

- 🩸 **自动生成线索（未实现）**  
  - 尸体  
  - 血迹  
  - 凶器划痕  

- 🦉 **主持人与场务操作**  
  - 可在案发后、查证前使用 **猫头鹰的法杖** 手动布置线索  
  - 使用法杖创建线索容器后，可用放大镜打开并放入证物  
  - 法杖还能高亮一定范围内的线索容器（因此玩家不应持有法杖）

---

#### 🦉 猫头鹰的法杖
- ✨ 能在任意方块上创建存储物品形式线索的容器  
- 🖐️ 拿在手上：高亮一定范围内（可配置）的线索容器方块  
- 🖱️ 左键：切换工作模式  
  - **create**：右键方块 → 将其变为容器  
  - **delete**：右键方块 → 删除该方块上的线索容器  

---

#### 🔎 放大镜
- ✨ 能够打开线索容器的存储界面  
- 🖐️ 拿在手上：当鼠标准心对准容器方块时提示玩家  
- 🖱️ 右键方块：如果该方块是线索容器，则打开其存储界面
